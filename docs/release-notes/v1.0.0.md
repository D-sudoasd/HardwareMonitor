# HardwareMonitor v1.0.0

First public release of HardwareMonitor, a low-obstruction Windows desktop micro-bar for live system telemetry.

## What's Included

- Always-on-top micro-bar for CPU, memory, disk usage, disk I/O, and temperature status.
- Click-to-expand trend panel with a 10-minute in-memory history.
- Portable Windows ZIP with `HardwareMonitor.exe`; no Python install is required for release users.
- LibreHardwareMonitor integration for CPU and disk temperature sensors.
- Explicit `N/A` handling for unavailable or invalid temperature sensors.

## Notes

Temperature support depends on hardware, firmware, Windows permissions, and LibreHardwareMonitor support. If a sensor returns invalid values such as `0.0 C`, HardwareMonitor shows `N/A` instead of reporting a false temperature.

Download `HardwareMonitor.zip`, extract it, and run `HardwareMonitor.exe`.
